# react-gift-react-foundation

## FE 카카오 선물하기 1주차 과제: React 기초

1. create-react-app 으로 프로젝트 생성 및 typescript로 동작되도록 세팅
2. tsconfig 설정
3. ESLint, Prettier 설정
4. emotion reset css 적용
5. gitignore 추가
6. 불필요한 코드 및 파일 제거
7. 폴더 생성

## 폴더 구조 설명

```bash
.root
├── node_modules
├── public
├── src
│   ├── assets          // 이미지, 폰트 등 미디어 파일
│   ├── components      // 주요 컴포넌트
│   ├── hooks           // 커스텀 훅
│   ├── pages           // 페이지 컴포넌트
│   ├── services        // api
│   ├── store           // 상태관리
│   ├── styles          // css
│   └── utils           // 공통함수, 상수 등
└──
```
